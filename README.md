# universal-llm-micro

Make me a LLM called Universal. This is a unique model because it uses really small custom LLMs for specific purposes then it uses those LLMs (btw make the small models from scratch called micro) to combine them into a model named Universal in pytorch and safetensor library

cd universal-llm-micro
npm i
```

npm run dev
```
